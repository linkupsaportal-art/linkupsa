# 🎨 خطة تنفيذ: إعادة تصميم صفحة استلام الطلبات بالكامل لتطابق لوحة التحكم (Cream & Lime)

خطة لإعادة تصميم صفحة استلام الطلبات (`/pickup`) بالكامل لتصبح تحفة فنية مبهرة بصرياً وتطابق نظام التصميم الراقي والحديث المتبع في لوحة التحكم (لوحة الألوان Cream & Lime مع التدرجات اللونية الفاخرة، الحواف المستديرة الفائقة، وتأثيرات الزجاج والميكرو-أنيميشن التفاعلية).

## 🛠️ التغييرات المقترحة

### 1. شاشة استلام الطلب الرئيسية ([page.tsx](file:///c:/Users/MSI-PC/OneDrive/Documents/freelancing/digital-delivery-platform/app/pickup/page.tsx))
- [ ] إدراج الكلاس الفخم `.theme-admin` لتفعيل نظام تصميم لوحة التحكم بالكامل (الخطوط، الألوان، الأزرار، وحقول الإدخال).
- [ ] استبدال الخلفية المسطحة بخلفية التدرج اللوني الفاخرة للوحة التحكم: `linear-gradient(180deg, hsl(170 8% 92%) 0%, hsl(186 11% 84%) 50%, hsl(190 12% 78%) 100%)`.
- [ ] إضافة إيقونة حزمة/طرد ثلاثية الأبعاد تنبض بالحياة مع ظل مشع فاخر بدلاً من الإيقونة التقليدية.
- [ ] إضافة لمسة أمان وعرض حالة الخادم مشعة مثل "بوابة الاستلام الآمن · متصل" مع وميض نابض.

### 2. نموذج البحث والاستعلام ([pickup-form.tsx](file:///c:/Users/MSI-PC/OneDrive/Documents/freelancing/digital-delivery-platform/app/pickup/pickup-form.tsx))
- [ ] إعادة تصميم الكارت بالكامل ليصبح بأسلوب زجاجي شفاف فاخر (`bg-white/90 backdrop-blur-xl border border-white/40 shadow-card rounded-3xl`).
- [ ] استخدام تأثير الظل الثقيل ثلاثي الأبعاد المعتمد في لوحة التحكم `.card-lift` ليعطي شعوراً بالفخامة والبروز.
- [ ] تحديث حقول الإدخال لتطابق شكل وملمس لوحة التحكم تماماً (إطار ناعم بلون الهايرلاين مع خلفية `bg-[hsl(var(--surface))]` وتأثير إشراق ناعم عند التركيز باللون الليموني المبهج).
- [ ] جعل مدخل "آخر 4 أرقام من الجوال" بتصميم متناسق تماماً ومحسّن للقراءة، مع الاحتفاظ بـ `dir="ltr"` لمنع أي التواء للنصوص.
- [ ] تحويل زر الإرسال إلى تحفة تفاعلية بلون الفوسفوري المكهرب (`bg-accent hover:bg-accent-hi text-accent-fg`) مع ظل ليموني مشع يزيد من روعة التصميم ومؤشر تحميل ناعم.

### 3. شاشة عرض تفاصيل الطلب وبيانات الحساب ([order-details.tsx](file:///c:/Users/MSI-PC/OneDrive/Documents/freelancing/digital-delivery-platform/app/pickup/order-details.tsx))

- [ ] تقسيم الحقول الخاصة ببيانات الدخول (البريد الإلكتروني، كلمة المرور، كود البطاقة) إلى كروت فرعية غاية في الأناقة بحدود ناعمة وخلفية بيضاء مع تأثيرات حركية عند التحويم.
- [ ] تجديد أزرار "النسخ السريع" لتصبح ليمونية ساطعة مع وميض عند النجاح وتغذية مرتدة فورية للمستخدم.
- [ ] إعادة تصميم شريط مؤشر الاستخدام الدائري أو التقدم (OTP limit indicator) بلون ليموني فاقع وخلفية غامقة ليصبح مبهجاً بصرياً.
- [ ] ترقية الملاحظات الهامة والتعليمات بصناديق تنبيهية حديثة تناسب التصميم العصري (باستخدام درجات الكهرمان والليمون المريحة للعين).
